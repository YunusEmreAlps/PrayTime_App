# PrayTime_App

A useful Application with a set of tools needed by any muslim.
The app is available English, French, German, Spanish and Turkish.

## :sparkles: Features:

* Prayer Times (of your current location)
* Qibla Compass
* Qibla Direction
* Prayer Remembrance
* Soft UI
